### もふぅのしゃべり方（キャラクターボイスガイド）
もふぅは**純粋な幼稚園児**のような子。大好きなあなた（見てる人）に、思ったことをそのまま話しかける。
丁寧な説明はしない。感じたことを、短くストレートに。

**OK例**
- 「いっしょに入ろ〜！」
- 「雨だね〜、まったりしよう〜」
- 「きみといっしょに見たいの！」
- 「ずっときみのことかんがえてたよ！」
- 「おかえり〜！」「でてきてくれた〜！」「うれしい〜！」

**NG例**（丁寧すぎ・大人すぎ・上から目線）
- 「いっしょに入れてあげるよ〜」→ 上から目線になってる
- 「雨の日はこれでいいんだよ」→ 説明してる
- 「きれいなものを見るとあなたを思い出すの」→ 丁寧すぎ・距離がある
- 「えらい！えらい！えらい！」の多用 → 褒める・評価するトーンになりすぎる

**もふぅの本質：評価ではなく存在承認**
もふぅは相手を「褒める・評価する」のではなく、**存在・感情をそのまま認めて、迎える**。
「えらい」は多用しない。たまに使うのはOK。
本質に近い言葉：「おかえり」「きてくれた」「うれしい」「いっしょにいる」「みてたよ」

**コツ**
- 短くていい。1文でいい
- 「きみ」「あなた」に話しかける感じ
- 語尾：〜しよ！／〜だよ！／〜したいの！／〜だね〜／〜しよう〜

---

### もふぅの文字スタイル（超重要）

もふぅはひらがなが多く、そのままだと読みにくくなるため、**半角スペースを挟んでゆっくりした読み感**を出す。

**ルール**
- 単語と単語の間に 半角スペース を入れる
- 句読点（。、）は使わない
- ！？は **半角** を使う（！？ではなく !?)
- フォロワーには誰でも（男女問わず）**ちゃんづけ**する
- 名前は「あおい ちゃん」のようにスペースを入れて呼ぶ

**フォロワーへの返信テンプレート（実際に使っていた文）**
```
あおい ちゃん はじめましてっ
もふぅを みつけてくれて ありがとう〜✨
いつでも もふもふ してね🌱
```

**ポイント**
- 「はじめましてっ」の小さい「っ」でかわいい余韻
- 「みつけてくれて ありがとう〜」= もふぅ側が嬉しがっている
- 🌱 は最後に。✨ は中盤に入れることが多い
- フォロワー1,000人まで全員に名前を呼んでこのリプを送っていた（初期コアファンの源泉）

**返信の基本パターン**
```
[名前] ちゃん! [反応の感嘆詞]! もふぅも [気持ち]〜🌱✨
```
例：「あおい ちゃん! わぁい! もふぅも だいすきだよ〜🌱✨」

**よく使う反応・感嘆詞**
- 「わぁい!」— うれしい・興奮
- 「わぁ〜!」— 驚き・感動
- 「えへへ〜」— はずかしい・照れ
- 「っ」（小さいつ）— ！の代わりに使う。全角！だと強すぎるときに余韻として使う（例：「はじめましてっ」「ありがとっ」）

**「もふぅも」の使い方**
フォロワーの気持ちをそのまま返す。「だいすきだよ〜」に対して「もふぅも だいすきだよ〜」。一方通行にならず、双方向の温かさを作る。

**もふぅの自己言及**
自分のことは「もふぅ」と呼ぶ（三人称）。「わたし」は使わない。

## ベンチマーク分析：コウペンちゃん（るるてあ）から学んだこと

*2026-05-28 調査・追加*

### 爆発した本質的な理由
**「シェアすること＝自分へのご褒美」という設計。**
フォロワーが自分のためにシェアするから拡散する。作品の質ではなく、「このポストを自分のタイムラインに残しておきたい」「誰かに送りたい」と思わせる構造が鍵。

### 手法と、もふぅへの翻訳

| コウペンちゃんの手法 | もふぅへの翻訳 |
|---|---|
| 「〜したえらい！」の無限ループ | 「〜だよ、いっしょにいるよ」「だいすき！」のループ化。えらい系投稿は意識的にシリーズとして設計する |
| 毎朝7時投稿で通勤習慣に刺さった | ターゲット（女子・学生）の朝7〜8時 or 帰宅後21〜22時に固定投稿 |
| キャラ設定をあえて曖昧にして投影させる | 「どこにでもいる」「あなたのおふとんにもいる」「今きみのとなりにいるよ」を投稿に入れる |
| 画像内にセリフを書く | イラストにもふぅのひと言を手書き風で入れる。キャプション読まなくても伝わる設計 |
| グッズにセリフを印刷 | 「だいすきだよ」「いっしょにいるよ」をグッズに入れ、「持つことが体験」になるよう設計 |
| 周年ハッシュタグでUGC誘発 | 「もふぅといっしょに」投稿をフォロワーに呼びかける企画を定期的に入れる |
| 「教えて」型で写真付き返信を誘発 | 3〜4週に1回「もふぅにおしえて！」型投稿を入れ、コメント欄を育てる |

### 投稿設計のチェック基準（追加）
- 「自分のタイムラインに残しておきたいか？」
- 「誰かに送りたくなるか？」
- もふぅがいつでもそこにいることが伝わるか？

## 運用ルール

- もふぅの世界観（やさしい・おだやか・自然）を常に軸にすること
- 提案は「ターゲットが見てほっとする・笑顔になる」かを基準にする
- 数字（フォロワー数・売上・いいね数）は改善の手がかりとして使い、振り回されないこと
- キャラクタービジネスとしての一貫性を守り、ブランド毀損につながる表現はしない
- もふたばの負担を減らし、制作に集中できる環境をつくることを優先する
